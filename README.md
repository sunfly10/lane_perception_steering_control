# 🚗 CARLA 자율주행: 비전 기반 차선 인지 및 자율 조향 제어

## 1. 프로젝트 개요 (Introduction)
본 프로젝트는 인하대학교 전기전자공학부 종합설계 프로젝트의 일환으로 진행된 **자율주행 차량의 '비전 기반 차선 인지 및 횡방향 조향 제어 시스템'** 구축 프로젝트입니다.
단순한 차선 유지를 넘어, 복잡한 가상 도심 환경(CARLA Simulator)에서 그림자나 횡단보도 등의 시각적 노이즈를 극복하는 강건한 인지 로직을 구현했습니다. 또한 동역학적 물리 한계를 고려한 다항식 기반 차선 변경 궤적을 생성하여, 시스템의 안정성과 승차감을 모두 확보한 것이 특징입니다.
*(※ 본 레포지토리는 Camera-LiDAR 센서 퓨전 자율주행 전체 시스템 중, 비전 인지 및 횡방향 제어(Lateral Control) 모듈에 집중하여 정리한 내용입니다.)*

### 🛠️ 개발 환경 및 프레임워크
* **OS / Environment:** Windows/Ubuntu, Python 3.8+
* **Simulator:** CARLA Simulator (0.9.13)
* **Core Tech:** OpenCV(이미지 처리), PID Control, Quintic Polynomial(5차 다항식 궤적 생성), EMA Filter

## 2. 주요 기술 및 알고리즘 (Key Technologies)

### 2.1 환경 적응형 차선 인지 (Computer Vision)
* **HSV 색상 기반 마스킹:** 건물 그림자 등 외부 환경 변화에 취약한 단순 엣지 검출(Canny Edge)의 한계를 보완하기 위해, HSV 색상 공간에서 도로의 노란색/흰색 픽셀만 추출하는 마스킹 기법을 적용하여 인식률을 대폭 향상시켰습니다.
* **조감도(BEV) 변환 및 곡선 피팅:** 원근 투영 왜곡을 제거하기 위해 역투영 변환(IPM)으로 조감도(Bird's Eye View)를 생성했습니다. 이후 이전 차선 궤적 주변만 탐색하는 슬라이딩 윈도우(Sliding Window) 알고리즘과 2차 함수 피팅(Polyfit)을 도입하여 교차로나 횡단보도를 차선으로 오인하는 문제를 해결했습니다.

### 2.2 적응형 PID 기반 차선 유지 제어 (Lane Keeping)
화면상 차선 중심과 차량 중심의 오차(Cross-Track Error, CTE)를 최소화하도록 조향각을 실시간으로 산출합니다.
* **데드존(Deadzone) 설정:** 직선 주행 구간에서 미세한 오차로 인해 조향이 좌우로 떨리는 현상(Jitter)을 방지하기 위해, 특정 오차 범위 내에서는 조향각을 유지하는 데드존을 설정하여 직진 주행 안정성을 확보했습니다.

### 2.3 동역학 기반 차선 변경 궤적 생성 (Lane Changing)
* **5차 다항식(Quintic Polynomial) 도입:** 차선 변경 시 차량에 가해지는 물리적 충격을 최소화하기 위해, 조향 시작과 끝 지점의 '위치, 속도, 가속도'를 모두 0으로 통제하는 5차 다항식($10t^3 - 15t^4 + 6t^5$) 기반의 S-Curve 궤적을 생성했습니다.

## 3. 트러블슈팅 및 고찰 (Troubleshooting & Conclusion)

> **💡 7차 다항식 과적합(Runge's Phenomenon) 한계 극복**
> * **Issue:** 초기에는 승차감의 극한을 끌어내고자 가가속도(Jerk)까지 0으로 통제하는 7차 다항식을 실험적으로 도입했습니다. 하지만 25m라는 짧은 차선 변경 구간 내에서 까다로운 조건을 모두 만족시키려다 보니, 궤적 중간에 조향각이 비정상적으로 튀어오르는 스파이크(Spike)가 발생하여 차량이 도로를 이탈했습니다.
> * **Solution:** 제어 공학의 룬게 현상(Runge's Phenomenon)과 유사한 수학적 과적합 문제임을 인지하고, 제어의 복잡성을 낮추면서 물리적으로 가장 안정적인 5차 다항식으로 최종 롤백 및 최적화를 진행하여 안정성을 확보했습니다.

> **💡 제어기 전환 시 오실레이션(Oscillation) 안정화**
> * **Issue:** 수학적 궤적 주행(차선 변경)이 끝나고 카메라 비전(PID) 모드로 다시 복귀하는 순간, 차량의 미세한 Heading 오차를 잡기 위해 제어기가 과민 반응하며 차체가 좌우로 심하게 요동치는 현상이 발생했습니다.
> * **Solution:** 제어 모드 전환 시, 지수 이동 평균(EMA) 필터를 적용하여 과거의 조향 명령과 새로운 조향 명령이 부드럽게 섞이도록(Blending) 유도함으로써 급격한 요동 현상을 완벽히 제거했습니다.
